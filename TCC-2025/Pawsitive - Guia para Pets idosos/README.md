# Trabalho de conclusão de curso — A elaboração de um guia online para auxiliar no cuidado de cães e gatos deficientes e idosos na região de Jundiaí.

Bem-Vindo(a) ao pawsitive, aqui voce podera ver um projeto desenvolvido pelas alunas 3º ano do ensino médio integrado ao técnico em informática para internet da Escola Técnica Estadual de Campo Limpo Paulista (ETECAMP).

# 🐕‍🦺Objetivos e problematica:
Nosso sistema nasce de diversas pesquisas realizadas que mostram a necessidade de oferecer informações claras e confiáveis para tutores de animais idosos e/ou com deficiência, que muitas vezes têm dificuldade em encontrar orientações específicas na internet. Percebemos que ainda existe pouca informação de qualidade sobre esse tema, o que torna nosso projeto ainda mais importante.

Nosso objetivo é criar um ambiente confiável para os tutores, onde eles possam acessar conteúdos seguros para se orientar e esclarecer dúvidas. Além disso, buscamos ajudar a reduzir a falta de informações disponíveis sobre animais idosos e/ou com deficiência, oferecendo materiais que realmente possam apoiar quem cuida deles.

O guia conta com uma parte informativa, abordando os cuidados básicos, alimentação adequada e adaptações comuns para diferentes tipos de deficiência tanto para gatos, quanto para cachorros. Além disso, inclui uma seção voltada para adoção e uma lista de locais onde os tutores podem encontrar produtos e profissionais especializados.

# 🦴Como executar o projeto:
1 - Entre no link a seguir para baixar o Visual Studio Code: https://code.visualstudio.com/ (Baixa o pacote base para seu sistema e siga os passos de instalação)

2 - Entre no link para instalar o xampp: https://www.apachefriends.org/pt_br/download.html (Baixa o pacote base para seu sistema e siga os passos de instalação)

3 - Entre no seu explorar de arquivos, vá em este computador, entre em seu hd principal, entre em xampp, entre em htdocs, baixe o arquivo do sistema pawsitive e descompacte ele dentro do htdocs.

4 - Abra o seu control panel do xampp e ative o MySQL e Apache.

5 - Depois após MySQL e Apache ficarem verdes, iremos abrir o navegador com duas guias. Em uma, deve-se pesquisar por “localhost/phpmyadmin” para importarmos o banco de dados.

6 - Assim que abrir o phpmyadmin, vamos criar um novo banco com o nome que esta salvo na pasta da aplicação(pawsitive).

7 - Vá ate a parte de "importar" e selecione o arquivo .sql que esta junto com a pasta do sistema.

8 - Agora basta pesquisar por "“localhost/nome do arquivo" em seu navegador, ir ate a pasta frontend e selecionar pginicial.php.
