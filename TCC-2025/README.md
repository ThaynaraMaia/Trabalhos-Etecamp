Nesse repositório possui uma cópia da parte prática dos Trabalhos de conclusão de curso (TCC) realizados na ETEC de Campo Limpo Paulista no ano de 2025.

- Desenvolvimento de um website para auxiliar no planejamento pessoal financeiro dos estudantes do ensino médio da ETEC de Campo Limpo Paulista (Finantec)

- Desenvolvimento de uma plataforma para auxiliar jovens brasileiros na busca por bolsas de estudos internacionais de graduação (CiaoMundo- BOLSAS DE ESTUDOS)

- Projeto Martopia: desenvolvimento de um sistema web para auxiliar na educação de jovens do interior de São Paulo sobre a vida marinha do litoral paulista (Martopia - vida marinha)

- Desenvolvimento de um website para praticantes de esportes em Campo Limpo Paulista (Praticantes de Esportes)

- A elaboração de um guia online para auxiliar no cuidado de cães e gatos deficientes e idosos na região de Jundiaí (Pawsitive - Guia para Pets idosos)

- Desenvolvimento de uma plataforma web para auxiliar na gestão documental e folha de pagamento do departamento pessoal em empresas de médio e pequeno porte (CONECTHAR- Dep. Pessoal em empresas)

- Implementação de uma plataforma digital para auxiliar no apoio emocional de jovens LGBTQIAPN+ na região de Jundiaí (JOVENS LGBTQIAPN+)

- Desenvolvimento de um website para auxiliar na adaptação pessoal de pessoas adultas diagnosticadas tardiamente com TEA (Adultos com TEA)

Os trabalhos são encontrados no [Repositório Institucional do Conhecimento - RIC-CPS](https://ric.cps.sp.gov.br/)
