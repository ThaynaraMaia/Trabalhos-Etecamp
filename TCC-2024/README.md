Nesse repositório possui uma cópia da parte prática dos Trabalhos de conclusão de curso (TCC) realizados na ETEC de Campo Limpo Paulista no ano de 2024.

Desenvolvimento de um museu virtual para a ETEC de Campo Limpo Paulista (MuseuVirtual)

Desenvolvimento de um sistema web para auxiliar na organização da cantina escolar da ETEC de Campo Limpo Paulista (Cantina_Escolar)

Identidade visual e marketing: justificativas e proposta para o redesign de uma estética automotiva (Estética_Automotiva)

Desenvolvimento de um sistema web para auxiliar na organização da semana Paulo Freire (TCC-SPF)

O desenvolvimento de uma rede social para o cenário de e-sports (E-sporting_teamPlay)

Implementação de uma plataforma digital na divulgação de trabalhadores autônomos da região de Jundiaí (Trabalhadores_Autônomos)

Sugestões de hobbies para o controle de ansiedade e estresse na rotina diária dos jovens (Hobbies)

Guia para auxiliar no bem-estar psicológico de pessoas com transtornos de bipolaridade (Transtornos_de_Bipolaridade)

Desenvolvimento de uma plataforma digital destinada a promover as ações do grupo escoteiro João Paulo II (Escoteiro)

Desenvolvimento de uma plataforma digital para o preparo de estudantes para o ENEM (Preparo_para_o_Enem)

Guia para auxílio de mães adolescentes de primeira viagem (Guia_para_mães_adolecentes)

Os trabalhos são encontrados em https://ric.cps.sp.gov.br/
